# Dispersion Analyzer — Milestones

## Milestone 1 — Group Creation & Calibration

* Image import (camera / gallery)
* Manual calibration using reference length
* Aiming point placement
* Manual marking of dispersion points
* Coordinate extraction relative to aim point
* Save a single dispersion group locally

## Milestone 2 — Dataset Management & Listing

* List all created dispersion groups
* View basic group metadata (distance, shot count, calibration)
* Delete / rename groups
* Persist datasets locally (no cloud)

## Milestone 3 — Combine Datasets

* Select multiple groups
* Combine selected groups into a new dataset
* Preserve source group references
* Display combined dataset as a single dispersion set

## Milestone 4 — Dataset Comparison & Mann–Whitney U-test

* Dataset comparison screen
* Select two datasets (single or combined)
* Mann–Whitney U-test implementation (pure Dart)
* Display U value, p-value, and significance result
* Visual comparison on dispersion graph

## Milestone 5 — Outliers, Validation & Final Polish

* Outlier flagging / highlighting
* Validation against provided Excel reference
* Minor UI polish for clarity and usability
* Bug fixes and final delivery build
