# Flutter 3.44.8 toolchain pin

Bump repo Flutter pin from `3.44.7` → `3.44.8` (Dart stays `3.12.2`).

## Why

Local/SDK stable is already 3.44.8; CI/docs/README badges still said 3.44.7.

## How

```bash
python3 tool/update_agent_toolchain_versions.py --from-sdk
```

Synced sinks: `docs/toolchain_versions.env`, README badges, `docs/tech_stack.md`,
CI workflows (`ci`, `dependency-updates`, `drift`, `deploy_web`), Melos baseline
header.

## Fallout

`apps/mobile/ios/Runner/SceneDelegate.swift`: `AppLinks.shared.handleLink(url:)`
now returns `Void` (app_links 6.4.x). Dropped Bool success check that broke
iOS simulator Xcode build on 3.44.8.

## Verification

- `./bin/checklist` green
- `./bin/integration_tests` on iPhone simulator + chrome preflight green
